# Áreas 3D — Galería

Una galería web moderna e interactiva de áreas y modelos 3D con diseño futurista y animaciones fluidas.

## 🎨 Características

- **Diseño Futurista**: Interfaz moderna con tema oscuro y colores neón (azul cian y naranja)
- **Galería Interactiva**: Grid responsivo con efectos hover y animaciones suaves
- **Filtros Dinámicos**: Sistema de pestañas para filtrar galerías por categorías
- **Modal Interactivo**: Vista detallada de elementos con descripciones
- **3D Elements**: Cubo animado en la navegación y elementos con perspectiva 3D
- **Animaciones**: Partículas flotantes, pulsos de brillo y transiciones suaves
- **Responsive**: Diseño adaptable a dispositivos móviles y escritorio
- **Integración WhatsApp**: Botones CTA para contacto directo

## 🛠️ Tecnologías

- **HTML5**: Estructura semántica
- **CSS3**: Estilos avanzados con animaciones y transformaciones 3D
- **JavaScript**: Interactividad y dinámica (implementación en archivo separado)
- **Google Fonts**: Tipografías Orbitron y Rajdhani

## 📋 Estructura

```
├── index.html          # Archivo principal con HTML y estilos
└── README.md           # Esta documentación
```

## 🎯 Secciones

1. **Navegación Sticky**: Barra superior con logo animado y enlaces
2. **Hero Section**: Encabezado con título principal y botones de acción
3. **Galería**: Grid de elementos con thumbnails
4. **Filtros**: Botones para categorizar contenido
5. **Modal**: Ventana emergente con detalles del elemento seleccionado
6. **CTA WhatsApp**: Sección de llamada a la acción
7. **Footer**: Pie de página con enlaces

## 🚀 Uso

1. Abre `index.html` en tu navegador web
2. Navega por la galería usando el grid
3. Usa los filtros para ver categorías específicas
4. Haz clic en cualquier elemento para ver detalles en el modal
5. Contacta por WhatsApp con los botones disponibles
